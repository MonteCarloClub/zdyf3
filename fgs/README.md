## 细粒度权限管理-前端

pnpm + vue3 + [antdv](https://antdv.com/components/overview)

### 初始化（安装依赖）

```bash
$ pnpm install
```

### 开发

```bash
$ pnpm start
```

### 打包

```bash
$ pnpm run build
```
直接打包在 `../backend/src/main/resources/fgs` 下

### Lints and fixes files

```bash
$ pnpm run lint
```
