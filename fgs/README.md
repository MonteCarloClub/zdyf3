## 细粒度权限管理-前端

pnpm + vue2

### 初始化（安装依赖）

```
pnpm install
```

### 开发

```
pnpm start
```

### 打包

```
pnpm run build
```
直接打包在 `../backend/src/main/resources/fgs` 下

### Lints and fixes files

```
pnpm run lint
```
