## Abe-trustPlatform

这里是前端代码，pnpm + vue2

### 初始化（安装依赖）

```
pnpm install
```

### 开发

```
pnpm run serve
```

### 打包

```
pnpm run build
```
直接打包在 `../backend/src/main/resources/static` 下

### Lints and fixes files

```
pnpm run lint
```
